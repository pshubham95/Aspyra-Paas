Awesome Features

Use of Containers

Containers are lightweight virtualization environments. 6 to 12 times more number of containers can be deployed as compared to virtual machines on the same hardware

High Availability and Thin Provisioning

Through clever use of Docker Swarm packages and our own storage algorithm we have ensured high avalibility of the hosts as well as thin provisioning of resources.

Clean and Simple UI

We have designed a simple and user friendly GUI such that any user can easily make use of the different services provided by COEP Aspyra

Mobile and Desktop Appications

Along with Web UI we have also created an android application so that users can have access to their account from their Cell phones and Tablets.Also the users can use the Windows Desktop application to access the platform.
